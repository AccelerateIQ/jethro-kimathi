One-time setupConnect to Azure Storage

The admin saves blog posts and settings directly to Azure Blob Storage — no server needed. Set this up once.

  1. 1Azure Portal → your Storage Account → Shared access signature
  2. 2Services: Blob · Resource types: Object · Permissions: Read, Write, Create · Expiry: 2 years
  3. 3Click Generate SAS and copy the SAS token (starts with sv=)
